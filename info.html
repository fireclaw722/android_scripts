Fireclaw's Changes for LineageOS

Substratum Changes

OMS Commits

  • OMS commits for AOSP 8.1 taken from the Substratum Gerrit
    • frameworks/base changes 485, 487, 424, 448 do not merge cleanly
    • frameworks/base changes 500 and 501 are not included
    • packages/apps/Settings changes 473, 482, 447 do not merge cleanly

build property addition

  • OMS7 and the Substratum App considers LineageOMS community supported, so ro.lineageoms.version is set to ro.lineageos.version

Substratum App

  • The Substratum app is included as a pre-built application. APK taken from APKmirror, so the keys are unchanged so that the app can be updated from the Google Play Store

LineageOS Framework/SDK Changes

Trust

  • Merge change #217171 from the LineageOS Gerrit. This change shows whether vendor-level security patches are considered current or out-of-date in the Trust interface.

Battery Styles

  • Topic battery_customisation was merged from the LineageOS Gerrit.
    • The changes have been split and rebased onto AOSP/Lineage 9.0, so builds after Sept. 14, 2018 may not have those patches.

Android/AOSP Changes

MicroG Signature Spoofing Patch

  • Builds on/after Sept. 10, 2018 include omnirom's patches for Signature Spoofing.

EmojiOne

  • EmojiOne fonts replace AOSP default emoji
  • Builds before Sept. 6, 2018 include EmojiOne v3
  • Builds on/after Sept. 6, 2018 include EmojiOne v4

Battery info on Ambient Display

  • Ambient/Always-On display battery info ported from DU,AICP, and other ROM projects

System App Changes

Substratum

  • The Substratum app is included and updatable from the Play Store

F-Droid

  • F-Droid and it's companion Privileged Extension are included

Moto Widgets

  • Moto CommandCenter widget, included on stock Moto Z Droid devices, is included in the ROM
  • Moto TimeWeather, included with stock Moto G5 devices is included on builds that pre-date Aug. 14, 2018
  • Moto TimeWeather, included with the stock Z3 Play, is included on builds on/after Aug. 14, 2018

Microsoft Apps

  • Builds before Aug. 14, 2018 include Microsoft Office, Bing, Cortana, and other Microsoft Apps. Builds after do not include any.

Firefox

  • Firefox/Fennec replaces system default browser Jelly. Fennec taken from Mozilla's website and is updatable from the Play Store.
  • Firefox Klar, from F-Droid, is also included, and is updatable from F-Droid.

Device-Specific Changes

Efficiency Mode for Moto Mods

  • Merge topic moto-mods-battery-lineage-15.1 from the LineageOS Gerrit, for builds of devices that support Moto Mods. This includes:
    • Moto Z
    • Moto Z2 Force
    • Moto Z2 Play
    • Moto Z Play not included because the device tree does not properly support Moto Mods

SafetyNet commits

  • Devices that build have merged patches that bypass safetynet checks. Certain device(s) do not build with these commits, and they aren't included:
    • Moto Z Play

Work in Progress Devices

Certain Devices are built using unofficial repos and include unofficial changes. All other devices are built using official LineageOS and TheMuppets github repos. They are summarized below.

Moto Z Play

  • Built using BTbN's github repos
  • May sometimes include changes to support Moto Mods, and those changes may be removed at will.
  • Trust vendor patch level is set to 2018-04-01, which represents stock OPN27.76-12-22

Moto G4/G4 Plus

  • Built using sgspluss's github repos
  • Build Property changes for low RAM device, to run more smoothly on the 2GB RAM model.
  • Trust vendor patch level is set to 2018-04-01, which represents stock NPJS25.93-14.7-8
  • SELinux is set to permissive, otherwise it would not boot.
