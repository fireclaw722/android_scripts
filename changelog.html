Changes for Fireclaw's LineageOMS

Build types

  • Snapshot
    • Built when monthly security patches are merged. Exceptions are as follows:
      • critical device-specific bug is found, and solving requires re-build
      • LineageOS team provides patch to critical bug ahead of monthly security patch
      • device hasn't had a snapshot build and is ready for snapshot
    • -user builds, which don't permit SELinux permissive domains, nor allow root. It is recommended that you use Magisk for root if you plan on rooting.
    • Snapshot builds come from official LineageOS repos, and devices must meet the criteria for LineageOS official builds for snapshot
  • Experimental
    • Built including different patches to fix issues with OMS, Lineage, included system apps, or builds for devices that can't be built using -user builds
    • Features may quickly break and these builds will generally be less stable
    • Builds are not regular and may come at any time
