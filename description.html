Downloads and Updates

Downloads

All Downloads are located under /builds/.

Files under /images/ are only for testing purposes. They include the system, recovery, and boot images for flashing with fastboot, and aren't needed to use these builds.

Files under /updater/ includes folders for each device, and inside those folders contains .zip files that can be flashed in a compatible recovery, such as TWRP. It is recommended that you do a clean flash when coming from a different ROM or Stock. This means to wipe data and Internal Storage.

Updates

These builds use LineageOS's built in updater application, and on the server-side uses this PHP LineageOTA application. This means OTA updates work just like standard LineageOS.

Update Channels

There are two different update channels: STABLE and UNOFFICIAL.

The STABLE builds will be built after the monthly security patches are merged into LineageOS's official repos. This usually happpens near the beginning of the month.

The UNOFFICIAL is the regular update channel, and will be built at an irregular interval. Essentially, whenever I notice an interesting change get merged, or whenever I make a change to test, I will build an UNOFFICIAL version.
